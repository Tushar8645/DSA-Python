# Time and Space complexity of Stack operations with List

|                     | Time Complexity | Space Complexity |
| ------------------- | --------------- | ---------------- |
| Create Stack        | O(1)            | O(1)             |
| Push                | O(1) / O(n^2)   | O(1)             |
| Pop                 | O(1)            | O(1)             |
| Peek                | O(1)            | O(1)             |
| isEmpty             | O(1)            | O(1)             |
| Delete Entire Stack | O(1)            | O(1)             |
