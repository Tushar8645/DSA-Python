# Time and Space Comlexity of Doubly Linked List
<hr>

|                  | Time Complexity | Space Complexity |
| ---------------- | --------------- | ---------------- |
| Create           | O(1)            | O(1)             |
| Append           | O(1)            | O(1)             |
| Prepend          | O(1)            | O(1)             |
| Insert           | O(n)            | O(1)             |
| Search           | O(n)            | O(1)             |
| Traverse         | O(n)            | O(1)             |
| Reverse Traverse | O(n)            | O(1)             |
| Get              | O(n)            | O(1)             |
| Set              | O(n)            | O(1)             |
| Pop First        | O(1)            | O(1)             |
| Pop              | O(1)            | O(1)             |
| Remove           | O(n)            | O(1)             |
| Delete all nodes | O(1)            | O(1)             |
